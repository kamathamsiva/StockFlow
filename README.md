\# StockFlow



StockFlow is a full-stack inventory management application for managing products, stock levels, organizations, and user accounts.



\## Features



\- User signup and login

\- JWT authentication

\- Protected routes

\- Multi-organization data isolation

\- Dashboard with inventory statistics

\- Add, edit, delete, and search products

\- Low-stock detection

\- Organization settings

\- User profile

\- Responsive interface



\## Tech Stack



\### Frontend

\- React.js

\- React Router

\- Axios

\- Bootstrap

\- Vite



\### Backend

\- Node.js

\- Express.js

\- Sequelize ORM

\- JWT Authentication

\- bcrypt



\### Database

\- MySQL
